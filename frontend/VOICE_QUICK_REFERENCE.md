## Wake Word Mode

**With Wake Word (default: "vault"):**
1. Say "vault" to activate
2. Wait for "Listening" confirmation
3. Say your command
4. System returns to sleep after action

**Without Wake Word:**
- Commands work immediately
- No activation needed
- Disable in settings

## Troubleshooting

### Microphone Not Working
- Check browser permissions
- Ensure HTTPS connection
- Try different browser
- Check device microphone

### Commands Not Recognized
- Speak more clearly
- Check command list in settings
- Ensure wake word is correct
- Reduce background noise

### Mobile Issues
- Grant microphone permission
- Use supported browser
- Check device microphone
- Ensure stable connection
